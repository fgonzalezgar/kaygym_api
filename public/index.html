¿Cómo subir la imagen aquí?

  1. Guarda tu imagen (ej. logo.png) en la carpeta del backend:
    /public/images/
  2. Abre el archivo public/index.html (este archivo) desde tu editor de código.
  3. Borra el código del placeholder ("Tu Logo Aquí").
  4. Descomenta la etiqueta <img> que apunta a /images/logo.png.
